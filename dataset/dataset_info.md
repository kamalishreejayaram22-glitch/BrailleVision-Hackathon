# Dataset Information

## Overview

The dataset consists of Braille character images used for training and testing the YOLOv8 model.

## Classes

 A-Z English alphabet Braille characters
 
 Additional symbols and numbers (if applicable)

## Dataset Structure

Training Set

Validation Set

Test Set

## Annotation Format

YOLO annotation format was used for object detection training.

## Purpose

The dataset was used to train a YOLOv8 model for detecting Braille characters and converting them into readable English text.
